# 🛡️ MasterVPN (Windows) 

ابزاری قدرتمند برای دور زدن فیلترینگ و سیستم‌های DPI (Deep Packet Inspection) با استفاده از تکنیک **Domain Fronting**.
بدون نیاز به سرور یا کانفیگ پیچیده — فقط با یک اکانت گوگل، به اینترنت آزاد متصل شوید.

---

## ✨ قابلیت‌ها

* 🔹 **برنامه مستقل (`.exe`)**
  اجرای آسان بدون نیاز به نصب پایتون یا استفاده از ترمینال

* 🔹 **تزریق خودکار سرتیفیکیت**
  اضافه کردن هوشمند و نامحسوس MITM Certificate به ویندوز

* 🔹 **بدون محدودیت واقعی**
  مخفی شدن در ترافیک `www.google.com` با ریسک بسیار پایین برای بلاک شدن

* 🔹 **پینگ زنده و نودهای چرخشی**
  نمایش وضعیت اتصال + Load Balancing برای بهترین عملکرد

---

## 🚀 راه‌اندازی سریع (برای کاربران)

1. پروژه را دانلود کنید
2. فایل را Extract کنید
3. برنامه `MasterVPN.exe` را اجرا کنید
4. مقادیر زیر را وارد کنید:

   * **Google Script ID**
   * **Auth Key**
5. روی دکمه 🟢 **CONNECT** کلیک کنید
6. تنظیمات پراکسی سیستم یا مرورگر:

   ```
   Address: 127.0.0.1
   Port: 8085
   ```
7. 🎉 تمام! اینترنت آزاد در اختیار شماست

---

## 🛠 راه‌اندازی سرور (Google Apps Script)

بدون نیاز به VPS — اجرا کامل روی زیرساخت گوگل

### مراحل:

1. ورود به:

   ```
   https://script.google.com/
   ```

2. ایجاد پروژه جدید (New Project)

3. حذف کدهای پیش‌فرض

4. جایگذاری محتوای فایل:

   ```
   Code.gs
   ```

5. تنظیم مقدار:

   ```javascript
   const AUTH_KEY = "your_password_here";
   ```

6. Deploy کردن پروژه:

   ```
   Deploy → New Deployment
   ```

7. تنظیمات:

   * **Type:** Web app
   * **Execute as:** Me
   * **Who has access:** Anyone

8. روی **Deploy** کلیک کنید

9. مقدار **Deployment ID** را کپی کنید

---

## 🔐 اتصال به کلاینت

در نرم‌افزار ویندوز وارد کنید:

* `Google Script ID` ← Deployment ID
* `Auth Key` ← پسوردی که تعیین کردید
