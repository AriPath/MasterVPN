# 🛡️ MasterVPN (Windows)

> 🚀 ابزاری قدرتمند برای عبور از فیلترینگ و سیستم‌های **DPI (Deep Packet Inspection)**
> با استفاده از تکنیک **Domain Fronting** — بدون نیاز به سرور یا کانفیگ پیچیده

✨ تنها با یک اکانت گوگل، به اینترنت آزاد متصل شوید

---

## ✨ قابلیت‌ها

* ⚙️ **برنامه مستقل (`.exe`)**
  اجرای آسان بدون نیاز به نصب یا ترمینال

* 🔐 **تزریق خودکار سرتیفیکیت**
  اضافه کردن هوشمند MITM Certificate به ویندوز

* 🌐 **پنهان در ترافیک گوگل**
  استفاده از `www.google.com` برای کاهش احتمال بلاک شدن

* 🔄 **پراکسی خودکار ویندوز (جدید)**
  بدون نیاز به تنظیم دستی — با یک کلیک، کل سیستم تونل می‌شود

---

## 🚀 راه‌اندازی سریع

### 👤 برای کاربران نهایی

1. 📥 پروژه را دانلود کنید
2. 📦 فایل را Extract کنید
3. ▶️ برنامه `MasterVPN.exe` را اجرا کنید
4. 🔑 اطلاعات زیر را وارد کنید:

   * `Google Script ID`
   * `Auth Key`
5. 🟢 روی دکمه **CONNECT** کلیک کنید

✅ پراکسی ویندوز به صورت **خودکار** تنظیم می‌شود
❗ نیازی به وارد کردن `127.0.0.1:8085` نیست

🎉 **همین! اینترنت آزاد فعال شد**

---

## 🛠 راه‌اندازی سرور (Google Apps Script)

> ☁️ بدون نیاز به VPS — اجرا روی زیرساخت گوگل

### مراحل:

1. ورود به:

   ```
   https://script.google.com/
   ```

2. ➕ ایجاد پروژه جدید (New Project)

3. 🧹 حذف کدهای پیش‌فرض

4. 📄 جایگذاری محتوای فایل:

   ```
   Code.gs
   ```

5. 🔐 تنظیم پسورد:

   ```javascript
   const AUTH_KEY = "your_password_here";
   ```

6. 🚀 Deploy:

   ```
   Deploy → New Deployment
   ```

7. ⚙️ تنظیمات:

   * **Type:** Web app
   * **Execute as:** Me
   * **Who has access:** Anyone

8. ✅ کلیک روی **Deploy**

9. 📋 کپی مقدار:

   ```
   Deployment ID
   ```

---

## 🔗 اتصال به برنامه

در نرم‌افزار ویندوز وارد کنید:

* 🆔 `Google Script ID` ← Deployment ID
* 🔑 `Auth Key` ← پسوردی که تعیین کردید
